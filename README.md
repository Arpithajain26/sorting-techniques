Selection Sort works by dividing the array into two parts: a sorted subarray and an unsorted subarray.  
In each iteration, it selects the smallest element from the unsorted part and places it at the correct position in the sorted part.

**Time Complexity:**
- Worst Case: O(nÂ²)
